# 제품 개요

## 한 줄 설명

Mocha는 짧고 반복 가능한 모션 챌린지를 선택하고, 직접 시도하고, 결과를 확인할 수 있는 웹 기반 챌린지 플랫폼입니다.

## 핵심 사용자 흐름

1. 활성화된 챌린지를 탐색한다.
2. 챌린지 상세와 레퍼런스 프리뷰를 확인한다.
3. 로그인한 뒤 챌린지 시도를 시작한다.
4. 시도 영상을 업로드하고 처리 완료를 기다린다.
5. 결과와 개인 시도 이력을 다시 확인한다.

## 핵심 관리자 흐름

1. 관리자 계정으로 로그인한다.
2. 챌린지를 생성, 조회, 수정, 삭제하고 활성 상태를 전환한다.
3. 모델 자산을 업로드하거나 교체한다.
4. 챌린지 기준 영상 분석을 실행한다.

## 현재 제품 경계

- 웹 MVP 우선
- 유사도 기반 점수 제공
- 소셜 기능보다 챌린지 참여와 이력 관리 우선
- 현재 단계에서는 세션 기반 인증으로 충분

## 현재 비목표

- 네이티브 모바일 앱
- 실시간 멀티플레이
- 소셜 피드, 팔로우 그래프, 크리에이터 마켓플레이스
- `USER`, `ADMIN`을 넘는 복잡한 권한 체계

## 제품 방향

- 챌린지 참여 흐름은 단순하게 유지한다.
- 관리자 기능은 명시적이고 CRUD 중심으로 설계한다.
- 공개 사용자 UX와 관리자 UX를 분리한다.
- 회원 소유 데이터와 프로필 기능은 작은 단위로 확장한다.
