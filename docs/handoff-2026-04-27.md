# Mocha Handoff 2026-04-27

## 이번 UI/UX 마무리 작업 요약

### 프론트 전체 방향:

- 홈/내부 페이지 헤더 UI를 최대한 통일.
- 노란 포인트 컬러 #ffdd8a 중심으로 CTA, 활성 메뉴, hover 색감 정리.
- 전체 radius를 대부분 8px 수준으로 낮춰 둥근 느낌을 줄이고 더 단단한 UI로 변경.
- 불필요한 그림자/그라데이션/과한 효과를 줄여 성능 부담 완화.

### 헤더:

- 내부 페이지 헤더 폭을 홈과 맞춰 1320px 기준으로 정리.
- 홈 헤더도 내부 헤더와 같은 구조로 변경:
Mocha / 홈·챌린지·게시판 / 마이페이지 또는 관리자·로그아웃
- 닉네임 환영문구는 제거된 상태 유지.
- 내부 헤더 네비도 홈과 같은 stage-nav stage-nav--landing 스타일 사용.
- 마이페이지, 관리자 버튼은 현재 페이지에서 메뉴처럼 노란 테두리/배경 활성화.
- 마이페이지/관리자/로그아웃 버튼은 기본 테두리 없이, hover 때만 노란 테두리 표시.
- 1100px 이하 내부 페이지는 모바일 하단 내비로 전환.
- 760px 이하 홈 헤더는 숨기고 하단 내비 사용.

### 홈:

- 모바일 히어로 이미지를 3:4 비율로 맞춤.
- 모바일 히어로 이미지와 CTA 가로폭을 맞춤.
- 쇼케이스/리뷰 캐러셀 hover 시 멈추지 않도록 변경.
- 쇼케이스/리뷰 영역 드래그바 안 보이도록 처리.
- 쇼케이스에도 노란 색감 적용.
- 리뷰 카드 하단 메타를 작성자 왼쪽, 날짜 오른쪽으로 분리.
- 풋터 색감/톤도 전체 UI와 맞춤.

### 게시판:

- 모바일 게시판 깨진 pseudo-content 라벨 수정:
조회, 댓글
- 모바일 리스트에서 상단 컬럼 제목은 숨기고, 행은 분류/고정 배지 + 제목만 표시.
- 조회/댓글/작성자/날짜 등은 상세 페이지에서 확인하는 방향으로 통일.
- 게시판 CTA 그룹은 모바일에서 우측 정렬되도록 정리.

### 마이페이지:

- 모바일 리스트는 상태/난이도 배지 + 제목만 표시.
- 점수/날짜/조회/댓글/좋아요 등은 상세 펼침 안에서 확인하도록 정리.
- 좋아요 탭은 난이도 / 제목 / 좋아요 중심으로 바꾸고, 상세 펼침 안에:
챌린지 보기, 삭제, 설명, 카테고리, 챌린지 시간 표시.
- 후기 리스트 별점 색상 노란색으로 변경.
- 좋아요 하트 색상은 챌린지 리스트와 맞춤.
- 회원탈퇴 버튼과 입력창 간격 조정.
- 모바일 CTA 우측 정렬 통일.

### 운영허브/내부 페이지:

- 1100px 이하에서 헤더 CTA 줄바꿈 대신 모바일 하단 내비로 전환.
- 모바일 CTA 정렬은 우측 기준으로 통일.
