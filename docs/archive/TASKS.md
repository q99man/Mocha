# 과거 작업 목록

이 문서는 과거 `TASKS.md`의 복구본입니다. 원본은 문자 인코딩이 심하게 손상되어 전체 항목을 그대로 유지하는 대신, 당시 누적되었던 작업 축을 읽을 수 있는 형태로 다시 정리했습니다.

## 당시 큰 작업 축

- React + TypeScript + Vite 프런트엔드 기반 구축
- Spring Boot + Gradle 백엔드 기반 구축
- MySQL, Redis, Docker Compose 로컬 실행 경로 정리
- 챌린지 목록/상세/API 흐름 구축
- 시도 업로드, 저장, 처리, 점수 반환 흐름 구축
- Motion session, runtime state, processing progress 계약 정리
- MediaPipe 브리지와 스텁 처리 경로 병행 지원
- 관리자 분석 화면과 운영 UI 보강
- UTF-8, BOM, 인코딩 문제 정리

## 현재 기준

- 이 문서는 과거 작업 흐름을 요약한 보관용 메모입니다.
- 현재 해야 할 일과 우선순위는 [../ROADMAP.md](../ROADMAP.md)를 기준으로 봐야 합니다.
