- 원인 추정: UI 문구가 한국어 기준으로 바뀌었지만 테스트 셀렉터가 이전 상태에 머물러 있습니다.

## 바로 다음 작업 추천

1. 프런트 테스트 고치기
- `AttemptsPage` 테스트에 `AuthProvider` 또는 이에 준하는 테스트 wrapper를 추가
- `AttemptResultPage` 테스트의 접근성 질의 문자열을 현재 UI 라벨에 맞게 갱신
- 가능하면 공통 `renderWithProviders` 유틸로 인증 컨텍스트를 묶어 재사용성 확보

2. 인증 흐름 회귀 점검
- `/auth` 진입 후 일반 사용자와 관리자 리다이렉트가 의도대로 동작하는지 확인
- `/admin/**` 직접 진입 시 비관리자 차단 동작 확인
- `AuthProvider` 초기 세션 로딩 중 화면 깜빡임이나 이중 요청이 없는지 점검

3. 관리자 챌린지 편집 UI 마무리
- 백엔드에는 `PUT /api/admin/challenges/{id}` 와 `PATCH /api/admin/challenges/{id}/active` 가 있으므로
- 프런트 관리자 화면에서 편집, 활성/비활성, 삭제 흐름이 실제로 모두 연결됐는지 확인 필요
